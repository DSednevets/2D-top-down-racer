# 2D Top-Down Racer

A simple 2D top-down racing game prototype written in **Python** using **pygame-ce**.

The project demonstrates a basic game loop, scene management, collision handling, and smooth player controls.

---

## Features
- Top-down 2D racing gameplay
- Smooth left/right steering
- Scrolling road with start banner
- Enemy traffic spawning with increasing difficulty
- Score tracking
- Menu and Game Over screens

---

## Requirements
- **Python 3.11+** (3.12 recommended)
- Windows (tested on Windows 10/11)

---

## Installation & Run (Windows)

```powershell
git clone https://github.com/DSednevets/2D-top-down-racer.git
cd 2D-top-down-racer
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
py topdown_racer\run.py
